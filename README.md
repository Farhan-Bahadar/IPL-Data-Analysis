# 🏏 IPL Data Analysis Project

## 📌 Overview
This project focuses on analyzing Indian Premier League (IPL) cricket data using Python and Data Analytics techniques. The main objective is to extract meaningful insights from IPL matches, teams, players, and performances through data visualization and statistical analysis.

The project helps understand:
- Team performances over seasons
- Player statistics
- Winning patterns
- Toss impact on matches
- Venue analysis
- Run trends and match insights

---

## 📂 Dataset
The dataset contains IPL match and ball-by-ball data collected from publicly available cricket databases.

### Files Used
- `matches.csv` → Match-level information
- `deliveries.csv` → Ball-by-ball data
